* 신고 페이지 테스트
브라우저 주소창에 아래 주소 입력
 - http://localhost:8091/report?type=auto&residentId=999
    ㄴ 자동 신고 테스트
 - http://localhost:8091/report?type=manual&residentId=999
   ㄴ 수동 신고 테스트
 - http://localhost:8091/report?type=auto&residentId=999&address=부산광역시 해운대구 달맞이길 117
   ㄴ 자동 신고 지정 주소지 테스트
 - http://localhost:8091/report?type=manual&residentId=999&address=서울 중구 세종대로 110
   ㄴ 수동 신고 지정 주소지 테스트
